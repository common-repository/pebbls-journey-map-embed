Easily embed customizable maps and journey stats from Pebbls into your WordPress site.

== Description ==
Showcase Your Travel Adventures with Pebbls Travel Tracker Map Embed plugin.

Pebbls Adventures is an exciting new and free platform designed for adventurers to track, map, and share their journeys. With the Pebbls Travel Tracker Map Embed plugin, you can seamlessly showcase your Pebbls Adventures maps directly on your WordPress site. This plugin allows you to embed interactive maps and journey stats, providing your audience with an engaging way to explore and follow your adventures as they happen.

On the road, deep in the mountains, or sailing across an ocean, Pebbls Travel Tracker Map Embed lets you showcase your adventures with stunning maps and stats.

### External Services

This plugin connects to the Pebbls Core API to fetch journey data for embedding maps within WordPress. The API is hosted on [Pebbls.com](https://www.pebbls.com).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pebbls-journey-map-embed` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Pebbls Journey Map Embed block in your Gutenberg editor to add your journey map.

== Frequently Asked Questions ==

= How do I get my API Key? =
You can obtain your API key by visiting [Get your API key](https://www.pebbls.com/embed-key/).

= My API key isn't working, what should I do? =
Ensure your API key is correct and has not expired. If the issue persists, contact support via the Pebbls website.

= Can I customize the map height? =
Yes, you can customize the map height directly from the block settings in the Gutenberg editor.
